The database


Webscrapper

The database of this website is populated by a webscrapper in Python that is scheduled in the crontab of the webserver.


The webscrapper runs daily and gathers the data from MassCash website.


Schema


The database table schema is:


" `draw_id` int(11) NOT NULL"

" `draw_date` date NOT NULL,"

" `draw_date_name` varchar(10) NOT NULL,"

" `win1` int(2) NOT NULL,"

" `win2` int(2) NOT NULL,"

" `win3` int(2) NOT NULL,"

" `win4` int(2) NOT NULL,"

" `win5` int(2) NOT NULL,"

" `ordseq` varchar(15) NOT NULL,"

" `jackpot` varchar(10) NOT NULL,"

" `winners` int(10) NOT NULL,"

" `location` varchar(255) NOT NULL,"

" PRIMARY KEY (`draw_id`)"
